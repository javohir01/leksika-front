# Leksika.uz

Leksika.uz is a dictionary application for English-Uzbek translations built using React.js.

## Features

- **English to Uzbek Translation**: Easily translate English words or phrases to Uzbek.
- **Search Functionality**: Quickly search for specific words or phrases.
- **Responsive Design**: The application is designed to work seamlessly across different devices.
- **Simple Interface**: User-friendly interface for easy navigation and usage.

## Installation

To run Leksika.uz locally, follow these steps:

1. Clone this repository:

   ```bash
   git clone https://github.com/yourusername/leksika-uz.git
   ```

2. Navigate to the project directory:

   ```bash
   cd leksika-uz
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm start
   ```

5. Open your browser and visit `http://localhost:3000` to view the application.
